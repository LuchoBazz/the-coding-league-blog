Algoritmo de Euclides

Implementación C++​

int gcd(int a, int b) {
    return (b == 0)?a:gcd(b, a % b);
}
int lcm(int a, int b) {  
    return (a * b)/gcd(a, b);  
}

Propiedades​

  • Dos números se llaman primos relativos, o coprimos, si su máximo común divisor es igual a 1.

    • gcd(a, b) == 1gcd\left(a,\:b\right)\:==\:1gcd(a,b)==1

  • Todo divisor común de aaa y bbb es divisor de gcd(a ,b)gcd(a\:,b)gcd(a,b).

  • gcd(a ,b)gcd(a\:,b)gcd(a,b) donde tanto aaa como bbb son distintos de cero, también se puede definir como el entero positivo más pequeño ddd que puede ser una solución/que se puede expresar como una combinación lineal de aaa y bbb en la forma d= a⋅p + b⋅qd=\:a\cdot p\:+\:b\cdot qd=a⋅p+b⋅q, donde tanto p como q son números enteros.

  • gcd(a, 0) = ∣a∣gcd\left(a,\:0\right)\:=\:\left|a\right|gcd(a,0)=∣a∣, para a ≠ 0a\:\ne \:0a=0, ya que cualquier número es divisor de 0, y el mayor divisor de a es ∣a∣|a|∣a∣.

  • Si a divide a b⋅cb \cdot cb⋅c y gcd(a,b)=dgcd(a,b)=dgcd(a,b)=d, entonces ad\frac{a}{d}da​ divide a ccc.

  • Si mmm es un número entero no negativo, entonces gcd(m⋅a, m⋅b) = m⋅gcd(a, b)gcd\left(m⋅a,\:m⋅b\right)\:=\:m⋅gcd\left(a,\:b\right)gcd(m⋅a,m⋅b)=m⋅gcd(a,b). También se sigue de esta propiedad que si gcd(a, b)=ggcd\left(a,\:b\right)=ggcd(a,b)=g, entonces ag\frac{a}{g}ga​ y bg \frac{b}{g}\:gb​deben ser coprimos. Trate de derivarlo usted mismo.

  • Si mmm es cualquier número entero gcd(a, b)=gcd(a+m⋅b, b)gcd\left(a,\:b\right)=gcd\left(a+m\cdot b,\:b\right)gcd(a,b)=gcd(a+m⋅b,b).

  • El GCD: gcd(a, b) = gcd(b, a%b)gcd\left(a,\:b\right)\:=\:gcd\left(b,\:a\%b\right)gcd(a,b)=gcd(b,a%b)

  • Si mmm es un divisor común positivo de aaa y bbb, entonces gcd(am, bm) = gcd(a, b)mgcd\left(\frac{a}{m},\:\frac{b}{m}\right)\:=\:\frac{gcd\left(a,\:b\right)}{m}gcd(ma​,mb​)=mgcd(a,b)​.

  • GCD es una función multiplicativa. Eso es si a1a1a1 y a2a2a2 son coprimos gcd(a1⋅a2, b)=gcd(a1, b)⋅gcd(a2, b)gcd\left(a1\cdot a2,\:b\right)=gcd\left(a1,\:b\right)\cdot gcd\left(a2,\:b\right)gcd(a1⋅a2,b)=gcd(a1,b)⋅gcd(a2,b). En particular, recordando que GCD es una función de valor entero positivo, obtenemos que gcd(a, b⋅c) = 1gcd\left(a,\:b⋅c\right)\:=\:1gcd(a,b⋅c)=1 si y solo si gcd(a, b) = 1gcd\left(a,\:b\right)\:=\:1gcd(a,b)=1 y gcd(a, c) = 1gcd\left(a,\:c\right)\:=\:1gcd(a,c)=1. si el gcd es uno, entonces no es necesario que sean coprimos para distribuir el gcd, además, cada gcd individualmente también debe ser 1.

  • El GCD es una función conmutativa: gcd(a, b) = gcd(b, a)gcd\left(a,\:b\right)\:=\:gcd\left(b,\:a\right)gcd(a,b)=gcd(b,a).

  • El GCD es una función asociativa: gcd(a, gcd(b, c)) = gcd(gcd(a, b), c)gcd\left(a,\:gcd\left(b,\:c\right)\right)\:=\:gcd\left(gcd\left(a,\:b\right),\:c\right)gcd(a,gcd(b,c))=gcd(gcd(a,b),c). Por lo tanto, gcd(a, b, c, ...)gcd\left(a,\:b,\:c,\:...\right)gcd(a,b,c,...) se puede usar para denotar el GCD de múltiples argumentos.

  • gcd(a, b)gcd\left(a,\:b\right)gcd(a,b) está estrechamente relacionado con el mínimo común múltiplo lcm(a, b)lcm\left(a,\:b\right)lcm(a,b): tenemos gcd(a, b)⋅lcm(a, b) = ∣a⋅b∣gcd\left(a,\:b\right)⋅lcm\left(a,\:b\right)\:=\:|a⋅b|gcd(a,b)⋅lcm(a,b)=∣a⋅b∣.

  • Las siguientes versiones de distributividad son válidas: gcd(a, lcm(b, c)) = lcm(gcd(a, b), gcd(a, c))gcd\left(a,\:lcm\left(b,\:c\right)\right)\:=\:lcm\left(gcd\left(a,\:b\right),\:gcd\left(a,\:c\right)\right)gcd(a,lcm(b,c))=lcm(gcd(a,b),gcd(a,c)) y  lcm(a, gcd(b, c)) = gcd (lcm(a, b), lcm(a, c))\:lcm\left(a,\:gcd\left(b,\:c\right)\right)\:=\:gcd\:\left(lcm\left(a,\:b\right),\:lcm\left(a,\:c\right)\right)lcm(a,gcd(b,c))=gcd(lcm(a,b),lcm(a,c)).

  • Si tenemos las factorizaciones en primos únicas de a = p1e1⋅p2e2⋅p3e3⋅⋅⋅pmema\:=\:p_1^{e_1}⋅p_2^{e_2}⋅p_3^{e_3}⋅⋅⋅p_m^{e_m}a=p1e1​​⋅p2e2​​⋅p3e3​​⋅⋅⋅pmem​​ y b= p1f1⋅p2f2⋅p3f3⋅⋅⋅pmfmb=\:p_1^{f_1}⋅p_2^{f_2}⋅p_3^{f_3}⋅⋅⋅p_m^{f_m}b=p1f1​​⋅p2f2​​⋅p3f3​​⋅⋅⋅pmfm​​ donde ei ≥ 0 e_i\:\ge \:0\:ei​≥0 y  fi ≥ 0\:f_i\:\ge \:0fi​≥0, entonces el gcd de aaa y bbb es:

  • Si tenemos las factorizaciones en primos únicas de a = p1e1⋅p2e2⋅p3e3⋅⋅⋅pmema\:=\:p_1^{e_1}⋅p_2^{e_2}⋅p_3^{e_3}⋅⋅⋅p_m^{e_m}a=p1e1​​⋅p2e2​​⋅p3e3​​⋅⋅⋅pmem​​ y b= p1f1⋅p2f2⋅p3f3⋅⋅⋅pmfmb=\:p_1^{f_1}⋅p_2^{f_2}⋅p_3^{f_3}⋅⋅⋅p_m^{f_m}b=p1f1​​⋅p2f2​​⋅p3f3​​⋅⋅⋅pmfm​​ donde ei ≥ 0 e_i\:\ge \:0\:ei​≥0 y  fi ≥ 0\:f_i\:\ge \:0fi​≥0, entonces el lcm de aaa y bbb es:
